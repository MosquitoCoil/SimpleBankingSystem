# Simple Banking System

<p align="center">
  <img src="Simple Banking System Screenshot/Main page.png" width="313" alt="Main Page" />
</p>

A **Simple Banking System** built with **JAVA, MySQL, NetBeans IDE 8.2 (Java Swing)**.  
The Banking System Application is  desktop-based software 
designed to manage and maintain bank accounts. It provides functionalities for creating accounts, checking balances, making deposits, and handling transactions securely and 
efficiently. The system interacts with a MySQL database to store and retrieve account 
information.

## Features
- Account Management: Create, update, and delete bank accounts.
- Balance Inquiry: Check the current balance of any account using its ID.
- Deposit Functionality: Add funds to a specified account.
- User Interface: Intuitive GUI using Java Swing for easy navigation.
- Database Integration: Uses MySQL to securely store account data.

---

## Technology Used
- Programming Language: Java
- GUI Framework: Java Swing
- Database: MySQL
- Development Tools: NetBeans
- Connectivity: JDBC for database communication
